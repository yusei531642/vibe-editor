# JetBrains Mono Nerd Font (bundled)

This directory contains the **Nerd Font patched** version of JetBrains Mono,
bundled with vibe-editor as the default terminal font.

## Files

- `JetBrainsMonoNerdFontMono-Regular.ttf` (Mono variant, weight 400)
- `JetBrainsMonoNerdFontMono-Bold.ttf`    (Mono variant, weight 700)

The `Mono` variant fixes icon glyphs to a single cell width so terminal
column alignment is preserved (Powerline / Devicons / Material Icons).

## Source

- Nerd Fonts: https://github.com/ryanoasis/nerd-fonts (release v3.4.0)
- Upstream JetBrains Mono: https://github.com/JetBrains/JetBrainsMono (v2.304)
