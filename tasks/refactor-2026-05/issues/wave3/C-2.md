TITLE: [perf] canvas: AgentNodeCard の teamMembersSig が drag 中も全 agent でフル走査
LABELS: performance,canvas,javascript

## 概要 / Summary

Canvas の `AgentNodeCard` 内で使われている `teamMembersSig` selector が、drag 中に毎フレーム全 agent カード × 全 nodes でフル走査・string concat を行っており、N² コストの perf hotspot になっている。30 カード × 60fps だと数千回/秒の selector 実行が発生し、ノード drag が体感的に重くなる。

## 根本原因 / Root cause

- `src/renderer/src/components/canvas/cards/AgentNodeCard/CardFrame.tsx:247-258` の `teamMembersSig` selector が `s.nodes` を全件走査して string concat を作る。
- xyflow は drag 中に毎フレーム `applyNodeChanges` で `nodes` 配列を新規生成する (新参照)。zustand selector は新参照を見て毎回再計算する。
- 全 AgentNodeCard が同じ selector を持つため、drag 1 frame あたり `N (cards) × N (nodes)` の文字列連結 → O(N²)。
- 30 cards × 60fps = 1800 sigs/sec、N=30 で 54k 文字列連結/秒。

## 再現手順 / Repro

1. Canvas で AgentNodeCard を 20-30 枚配置する。
2. その中の 1 枚を drag 移動する。
3. Chrome DevTools Performance パネルで record 中、 `AgentNodeCard` の teamMembersSig 計算が profile の上位に並ぶ (Self Time の合計が大きい)。
4. 低スペック端末では drag が visibly カク付く。

## 影響範囲 / Impact

- Canvas で agent カードを 10+ 枚扱うユーザー (multi-team / dual preset 利用者)
- drag のフレームレート低下 + CPU 上昇 (ファン回りや battery drain)
- severity: MEDIUM (perf regression、機能停止ではない)

## 修正方針 / Fix proposal

### 案1 (推奨): drag 中はシグネチャ計算を bailout

1. `useCanvasStore` 側で「drag 中フラグ」を持つ (xyflow の `onNodeDragStart` / `onNodeDragStop` で set)。
2. `teamMembersSig` selector を以下のようにラップ:
   ```ts
   const teamMembersSig = useCanvasStore((s) => {
     if (s.isDragging) return s._lastTeamMembersSig; // cached
     const sig = computeSig(s.nodes, teamId);
     s._lastTeamMembersSig = sig;
     return sig;
   });
   ```
3. もしくは AgentNodeCard 全体を `React.memo` で wrap し、drag 中は props 比較で bailout。

### 案2: teamMembersSig を canvas store の computed property に一元化

`useCanvasStore` に `teamMembersSigByTeam: Map<string, string>` を持ち、node 追加/削除/role 変更でのみ再計算。drag による position 変化では再計算しない (selector が dependency に入らない)。

### 案3 (簡易): `useShallow` / `subscribeWithSelector` で配列参照変化を吸収

zustand の `subscribeWithSelector` middleware と shallow equality を組み合わせ、teamMembersSig の入力が drag では変わっていないことを検知。
